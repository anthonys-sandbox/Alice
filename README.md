## GravityClaw

### Project Description
A lightweight, cross-platform tool for efficient file management and system automation. Built with Go and powered by a modular architecture.

### Usage Examples
```bash
# Basic file scan
gravity scan /path/to/directory

# Automated cleanup
gravity clean --verbose

# Generate report
gravity report --output=report.md
```
